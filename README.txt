J3MPI Rental Manager - Web Version
====================================

Works on: Windows, Mac, iPhone, iPad, Android â€” any device with a browser.

SETUP (do this once)
---------------------
1. Install Node.js from https://nodejs.org (LTS version)
2. Open a terminal / PowerShell inside this folder
3. Run: npm install
4. Run: npm start

ACCESS THE APP
--------------
When you run npm start, you will see something like:

  On this computer : http://localhost:3000
  On phone/tablet  : http://192.168.1.x:3000

- Open http://localhost:3000 on this computer
- Open http://192.168.1.x:3000 on your phone/iPad (must be on same WiFi)

Default login: admin / admin123

TO RUN AUTOMATICALLY ON WINDOWS
---------------------------------
Double-click START-WEB.bat

YOUR DATA
---------
Saved at: C:\Users\YourName\.j3mpi\rental.db
(Mac/Linux: ~/.j3mpi/rental.db)

NOTE
----
The server must be running on your PC for other devices to connect.
For access from anywhere (not just home WiFi), ask about online hosting.
